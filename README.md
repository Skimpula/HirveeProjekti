# HirveeProjekti
MökkiSovellus Maui C# Sql
This project uses:

- **.NET SDK version:** 8.0.419
- **Target frameworks:** 
  - net8.0-windows10.0.19041.0
  - net8.0-android
  - net8.0-ios
  - net8.0-maccatalyst
- **MAUI workload** is required (`dotnet workload install maui`)

How to run on Windows:
Open cmd on root of this project.
dotnet build
dotnet run -f net8.0-windows10.0.19041.0
